# Сценарий использования ** "Оформление заказа"**

**Цель:** оформление заказа товара на сайте

**Акторы:** зарегистристрированный пользователь, система (приложение и бекенд)

**Предусловия:** открыта форма заказа ПОМЕНЯЛАСЬ СТРОКА


## **Основной сценарий**

1. Пользователь открывает страницу "Пиццы"
2. Приложение открывает каталог пицц
3. Пользователь выбирает карточку товара (пиццы) и читает информацию
4. Пользователь добавляет товар в корзину
5. Приложение предлагает выбрать опции (размер, толщину теста, топпинги)
6. Пользователь выбирает опции
7. Приложение предлагает пользователю выбрать другой товар или перейти в корзину
8. Пользователь переходит в корзину
9. Пользователь заполняет дополнительную информацию в форме заказа (способ доставки и оплаты)
10. Пользователь отправляет заполненную форму
11. Приложение отображает статус приёма формы
